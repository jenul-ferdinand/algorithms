# FIT1008 Assignment 1 (Pocket Monsters)

This game's logic is based on Pokemon. The goal is to implement Pokemon Battles
functionality using Python with the given code scaffold. This assignment is testing
my knowledge of Python, Time Complexity and the use of Abstract Data Types/Structures.
(The use of Python built-in lists, dictionaries, sets, etc. is banned. But we can
use fixed size tuples)
