## All you need to know

This repository consists of the work I've done for
[FIT3155](https://handbook.monash.edu/2026/units/FIT3155?year=2026). Each week of work is organised into separate folders. The implementations have assertions within the file that test it. There are:

- C implementations _(run with `runc.sh <filename>.c`)_
    
    > You must have [tcc](https://bellard.org/tcc/) installed.

- Python implementations _(run with `python <filename>.py`)_

    > You must have [Python](https://www.python.org/downloads/) or [uv](https://docs.astral.sh/uv/getting-started/installation/) installed.

- TypeScript implementations _(run with `bun run <filename>.ts`)_

    > You must have [bun](https://bun.com/docs) installed.
