We have a outer iteration that allows us to insert all suffixes of S one by one, let this suffix string be S[i..n].

The thing that makes this polynomial is the fact that when inserting, we must always go compare down the path that matches the prefix of the suffix being inserted to create a new splitting internal node right before the mismatch.

So starting at the root node R, we check all outgoing edges. Starting from the first character match, let this edge's label be L, with a new loop to check if 
L[j..n] == S[j..n].

In the worst case, this edge label could match all the way up to i..n-1, that would make it O(n^2) in the worst case.
